# 示例

**简体中文** | [English](README.md)

演示 nix-devbox 用法的示例项目。

## 层次结构

```
┌─────────────────────────────────────────────────────────┐
│  第 3 层：你的项目                                       │
│  └── stacked-example/                                   │
│      → 覆盖 memory，添加 ports                           │
└─────────────────────────────────────────────────────────┘
                         ▲ 合并
┌─────────────────────────────────────────────────────────┐
│  第 2 层：应用配置                                       │
│  └── opencode/      → AI 助手，持久化数据               │
└─────────────────────────────────────────────────────────┘
                         ▲ 基于
┌─────────────────────────────────────────────────────────┐
│  第 1 层：基础配置                                       │
│  └── base/          → 通用 devshell 选项                │
└─────────────────────────────────────────────────────────┘
```

| 示例 | 层次 | 说明 |
|------|------|------|
| [base](base/) | 1 | 安全、资源、日志、tmpfs |
| [opencode](opencode/) | 2 | 卷挂载、环境变量、`$$` 转义 |
| [stacked-example](stacked-example/) | 3 | 展示配置层次堆叠 |

详细说明参见 [docs/configuration.zh-CN.md](../docs/configuration.zh-CN.md)。
